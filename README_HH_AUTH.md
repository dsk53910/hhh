# Инструкция: Как получить ключи для отклика на HH.ru

Чтобы бот мог не просто присылать вакансии, но и реально откликаться на них от вашего имени, ему нужен ваш **пользовательский токен** и **ID вашего резюме**.

## 1. Как получить HH_API_KEY (User Token)
Токен дает боту права действовать от вашего лица (подавать отклики).
*По правилам HH.ru, получение токена для личного использования требует создания приложения.*

1. Зайдите на [dev.hh.ru/admin](https://dev.hh.ru/admin) (войдите под своим логином HH).
2. Нажмите **Добавить приложение** (или Создать).
3. Заполните поля:
   - **Название:** `Мой личный Telegram бот`
   - **Описание:** `Для автоматизации личного поиска`
   - **Redirect URI:** `https://localhost`
4. После сохранения вы получите `client_id` и `client_secret`.
5. Вставьте свой `client_id` в эту ссылку и откройте её в браузере:
   `https://hh.ru/oauth/authorize?response_type=code&client_id=ВАШ_CLIENT_ID`
6. HH попросит вас подтвердить доступ. Нажмите "Разрешить".
7. Вас перекинет на пустую страницу `https://localhost/?code=XXX`. Скопируйте этот `code=XXX`.
8. Теперь нужно получить сам токен. Выполните этот запрос в терминале (подставив свои данные):
   ```bash
   curl -X POST -d "grant_type=authorization_code&client_id=ВАШ_CLIENT_ID&client_secret=ВАШ_CLIENT_SECRET&code=ВАШ_КОД" https://hh.ru/oauth/token
   ```
9. В ответ придет JSON с полем `access_token`.
10. Скопируйте этот токен и добавьте в файл `.env`:
    `HH_API_KEY="ваш_access_token"`

## 2. Как получить HH_RESUME_ID
Боту нужно знать, какое именно из ваших резюме отправлять работодателям.

1. Зайдите на сайт HH.ru и откройте свое резюме.
2. Посмотрите на URL-адрес в браузере. Он выглядит так:
   `https://hh.ru/resume/1234567890abcdef1234567890abcdef`
3. Скопируйте длинный код после `/resume/`. Это и есть ваш Resume ID.
4. Добавьте его в файл `.env`:
    `HH_RESUME_ID="1234567890abcdef1234567890abcdef"`

## 3. Перезапустите бота
После добавления ключей в `.env` выполните команду:
```bash
./hhh.sh stop
./hhh.sh start
```
Теперь при нажатии "Откликнуться" бот реально отправит ваше резюме!
